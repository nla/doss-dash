# doss-dash

This is a Dashboard for DOSS2
It draws pretty graphs using flot.


Building
--------

Requirements:

* Java 8
* Maven 3

Compile and run:
    
    mvn package && java -Ddoss.home=<DOSSHOMEDIR> -jar doss-dash-1.0-SNAPSHOT-jar-with-dependencies.jar
